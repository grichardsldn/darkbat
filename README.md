- create a symbolic link from /usr/local/include/X11 to /opt/X11/include/X11

# Viewer

```
$ cd viewer
$ make
$ ./view
```

Keys:
- h left
- j back
- k forward
- l right
- u up
- , down
- \- zoom out
- \= zoom in
